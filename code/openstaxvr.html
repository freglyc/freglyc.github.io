OpenStax VR Book Reader


OpenStax is a nonprofit company that creates educational tools to give students access to quality learning materials for little to no cost. The goal of the virtual reality book reader was to continue in this vein and to give students another potential learning resource through use of their mobile devices. VR goggles, which can be bought for as little as $15 or even created by hand, would be the only cost.


Developed over the course of a summer, this project was created in Unity and scripted in C# for both iOS and Android. While fully functional, this project is still a work in progress. You can view the source code on github.
